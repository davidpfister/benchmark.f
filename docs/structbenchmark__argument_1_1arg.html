arg
Argument

Inherits arg_base.

Definition

Represents the extended type for method argument .

Examples

use benchmark_argument
use benchmark_kinds
type(arg) :: a
a = arg(12.25_r8, 'arg1')
select type(x => a%value)
type is (real(r8))
!success
class default
stop
end select

Remarks

arg is the extended class of arg, which also contains bound procedures.
